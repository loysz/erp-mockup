# M&N Group ERP Mockup

Static ERP mockup for M&N Group, styled to resemble the ERPNext/Frappe Desk experience. This mockup is intended for early stakeholder review before the real ERP system is implemented in ERPNext.

## Preview

Open `index.html` in a browser to view the mockup locally.

## What's Included

- ERPNext-inspired workspace layout
- Advisory operations dashboard
- Client immigration tracker
- HR and payroll operations table
- Immutable audit trail mockup
- M&N logo asset
- Static HTML, CSS, and JavaScript only


## Deploy With GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to `Settings > Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save and wait for GitHub to publish the site.

Your mockup URL will look like:

```text
https://your-username.github.io/your-repo-name/
```

## Notes

This is a front-end mockup only. It does not include ERPNext backend logic, authentication, database storage, or document workflows yet.
